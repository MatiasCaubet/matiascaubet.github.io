Política de privacidad

Cuaderno Docente (Gestión de Cursos) · Vigente desde el 10 de junio de 2026

Cuaderno Docente es una aplicación para que docentes organicen sus cursos: asistencia, evaluaciones, notas y planificación de clases. Esta política explica qué datos trata la aplicación, con qué fin y qué derechos tenés sobre ellos.

Responsable del tratamiento

Matías Caubet — contacto: caubetmatias@gmail.com (Argentina).

Qué datos se recopilan

  • Datos de tu cuenta de docente: nombre, dirección de email y contraseña (almacenada únicamente como hash criptográfico; nunca en texto plano).
  • Datos que vos cargás para tu trabajo: escuelas, cursos, materias, nombres y apellidos de estudiantes, registros de asistencia, evaluaciones, notas, comentarios pedagógicos y materiales de clase (por ejemplo, PDFs).
  • Imágenes para importación (OCR): si usás la función de importar planillas desde una foto, la imagen que subís se procesa para extraer el texto.

La aplicación no recopila ubicación, contactos, ni accede a datos del dispositivo más allá de lo necesario para funcionar. No se muestra publicidad ni se usan identificadores publicitarios.

Datos de estudiantes

La aplicación está destinada a docentes (personas adultas). Los datos de estudiantes (nombre, apellido, asistencia y calificaciones) los carga el o la docente en ejercicio de su tarea profesional y bajo su responsabilidad. No se solicitan datos de contacto de los estudiantes, y esos datos nunca se usan con fines distintos a brindarle el servicio al docente.

Para qué se usan los datos

  • Prestar el servicio: guardar y mostrarte tus cursos, asistencias, notas y estadísticas.
  • Autenticarte y proteger tu cuenta (verificación de email, recuperación de contraseña).
  • Enviarte emails transaccionales (verificación, restablecimiento de contraseña). No se envía publicidad.

Tus datos no se venden ni se comparten con terceros con fines comerciales.

Servicios de terceros que procesan datos

Para funcionar, la aplicación se apoya en estos proveedores:

  • Hosting del servidor y base de datos (Railway): aloja la API y los datos del servicio.
  • Envío de emails (Resend): procesa tu dirección de email para entregar los correos transaccionales.
  • Reconocimiento de texto en imágenes (API de Anthropic): cuando usás la importación por foto, la imagen se envía a la API de Anthropic para extraer el texto (nombres y/o notas de la planilla). Según la política de la API de Anthropic, los datos enviados no se utilizan para entrenar sus modelos. La imagen no se usa para ningún otro fin.
  • Verificación de contraseñas comprometidas (Have I Been Pwned): al crear tu contraseña se consulta este servicio con un fragmento anónimo del hash (k-anonimato); tu contraseña nunca sale del servidor.

Seguridad

  • Toda la comunicación entre la app y el servidor viaja cifrada (HTTPS/TLS).
  • Las contraseñas se almacenan con hash bcrypt.
  • El acceso a tus datos requiere autenticación; cada docente solo puede ver sus propios cursos.
  • El servidor aplica límites contra intentos de acceso por fuerza bruta.

Conservación y eliminación

Tus datos se conservan mientras tu cuenta exista. Podés eliminar tu cuenta en cualquier momento desde la propia aplicación (Ajustes → Eliminar mi cuenta) o solicitándolo por email. Al eliminar la cuenta se borran de forma permanente todos los datos asociados: cursos, estudiantes, asistencias, evaluaciones, notas, comentarios y materiales. Más detalles en la página de eliminación de cuenta.

Tus derechos

Conforme a la Ley 25.326 de Protección de los Datos Personales (Argentina), podés ejercer los derechos de acceso, rectificación y supresión de tus datos escribiendo a caubetmatias@gmail.com. La Agencia de Acceso a la Información Pública (AAIP), en su carácter de órgano de control, tiene la atribución de atender denuncias y reclamos sobre incumplimiento de las normas de protección de datos personales.

Cambios a esta política

Si esta política cambia, se actualizará esta página y la fecha de vigencia. Los cambios sustanciales se comunicarán dentro de la aplicación.
